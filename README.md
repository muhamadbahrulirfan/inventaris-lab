## langkah instalasi
1. siapkan text editor seperti vs code
2. siapkan RDBMS mysql
3. nyalakan phpmyadmin kemudian upload file database inventaris_lab.sql
4. kalau sudah jalan program
5. untuk username dan password bisa lihat di tabel users

## tools yang digunakan dalam membangun sistem:
1. php native
2. mysql database
3. VS Code
4. HTML dan CSS
